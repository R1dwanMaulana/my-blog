## Berawal dari iseng-iseng 

Dunia Framework itu memang sangat menarik bagi kalangan developer khususnya javascript, jika kita bekerja dengan framework maka pekerjaan cepat selesai dan mudah untuk di maintenance.
Namun ada hal negatifnya juga, jika kita terlalu sering memakai framework, yaitu fundamental programmingnya akan hilang/lupa, padalah pondasi utama untuk memahami bahasa pemrograman yaitu basicnya dulu,
kalau basicnya tidak kuat, belajar framework pun akan kesulitan, karena antara program murni dengan framework, saling berkaitan.

Pengalaman saya ketika mencoba salah satu framework javascript yaitu Vuejs, saya masih mengalami kesulitan dalam pembuatan direktif, mungkin karena belum terbiasa,
maka dari itu saya harus membaca dokumentasinya berulang-ulang hingga paham life cycle dari vuejs tersebut. Setelah sekian lama, saya memutuskan untuk fokus kepada Vuejs, karena selera developer itu bead-beda.
